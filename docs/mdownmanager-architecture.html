// Security constraints
Scanner
Never routes to cloud. 3-layer. Auditable pattern library.
Conflict
Highest severity wins. Context may downgrade with shown justification only.
Critical

Egress
Explicit UI disclosure before any cloud send. Opt-in per feature.
